NAME
      mstatspop - Variability AnalysIs of multiple populations: Calculation and estimation of statistics and neutrality tests.

DESCRIPTION
      mstatspop v.0.1beta (20160308)
      Sebastian E. Ramos-Onsins, Luca Ferretti, Emanuele Raineri, Giacomo Marmorini, William Burgos and Gonzalo Vera
      Variability Analyses of multiple populations: Calculation and estimation of statistics and neutrality tests.
      

      Flags:
      -f, --format
            Input format file
            The possible values are:
                  fasta
                  tfa
                  ms

      -i, --input
            Path and name of the input file

      -o, --output
            Output format file
            The possible values are:
                  0 : extended
                  1 : single line / window
                  3 : dadi-like format
                  4 : single line pairwise distribution
                  5 : single line freq. variant per line/window
                  6 : SNP genotype matrix
                  10 : full extended

      -N    Populations
            Example: [#_pops] [#samples_pop1] ... [#samples_popN]

      Optional general parameters:
      -G    Last population is outgroup
            The possible values are:
                  0 : No (Default)
                  1 : Yes

      -u, --include-unknown
            Include unknown positions
            The possible values are:
                  0 : No (Default)
                  1 : Yes

      -T    Path and name of the output file
            Default value: Standard output

      -a    Alternative Spectrum File. Alternative_spectrum for each population (except outgroup)
            Example: File format: (average absolute values) header plus fr(0,1) fr(0,2) ... fr(0,n-1) theta(0)/nt, fr(1,1) fr(1,2) ... fr(1,n-1) theta(1)/nt...
            This option is used only for: Optimal Test

      -n    [Null Spectrum File: null_spectrum for each population (except outgroup)
            Example: (average absolute values) header plus fr(0,1) fr(0,2) ... fr(0,n-1) theta(0)/nt, fr(1,1) fr(1,2) ... fr(1,n-1) theta(1)/nt...
            This option is used only for: -a defined
            Default value: SNM

      -P    First value is the number of values to include, next are the number of lines to consider
            Example: ex: -P 6 1 2 4 8 16 64
            This option is used only for: Calculation of R2_p

      Optional parameters for FASTA and TFA input files:
      -O, --order
            Order of samples
            Example: [#_nsam] [number order of first sample, number 0 is the first sample] [second sample] ...etc. up to nsamples
            Default value: current order

      -t    Number of permutations per window (H0: Fst=0)
            Default value: 0

      -s, --seed
            Initial seed for random values
            Default value: 123456

      Parameters for TFA input files: (Sliding Window Analysis of Empirical Data)
      -w, --window-size
            Window size

      -z, --slide-size
            Slide size. Must be equal or larger than window size
            Default value: window size

      -Y, --window-length
            Define window lengths
            The possible values are:
                  0 : In 'effective' positions
                  1 : In 'physical' positions (Default)

      -W, --coords-file
            File with the coordinates of each window [init end] (overwrite options -w and -z)
            Default value: One whole window

      -E, --weights-file
            Input file with weights for positions:
            Example: Include three columns with a header, first the physical positions (1...end), second the weight for positions and third a boolean weight for the variant (eg. syn variant in nsyn counts is 0.000)
            Default value: all 1.000

      Parameters for MS input files: (Simulation Analysis of a Single Region)
      -l, --length
            Length

      -r, --ms-iterations
            Number of iterations
            This option is used only for: ms
            Default value: 1

      -m, --include-mask-file
            Include mask_filename
            Example: mask_file format: 1st row with 'length' weights, next sample rows x lengths: missing 0, sequenced 1)
            Default value: No mask. All positions included

      -v, --ratio-trans
            Ratio transitions / transversions
            Default value: 0.5

      -F, --include-outgroup
            Force analysis to include outgroup
            The possible values are:
                  0 : Ancestral
                  1 (Default)

      -q, --freq-reverted-mutation
            Frequency of reverted mutation
            This option is used only for: -F 1
            Default value: 0

      Parameters for FASTA input files: (Whole Region Analysis)
      -p, --ploidy
            Number of lineages per sequence
            The possible values are:
                  1 (Default)
                  2

      -g    GFF file
            Example: Add also: coding, noncoding, synonymous, nonsynonymous, silent, others (or whatever annotated). If synonymous or nonsynonymous or silent then add: Genetic_Code: Nuclear_Universal, mtDNA_Drosophila, mtDNA_Mammals, Other. If Other then introduce the code for the 64 triplets in the order UUU UUC UUA UUG ... etc.
            Default value: No annotation

      -c    Criteria to consider transcripts
            This option is used only for: Coding regions
            The possible values are:
                  max (Default)
                  min
                  first
                  long

      Help:
      -h, --help
            Help and exit
