For the calculation mstatspop, formats allowed:


  • Fasta: Using the fasta file it is only possible to make a whole analysis of the entire region but not a sliding window analysis.
  • MS: The calculation mstatspop allow to include ms format files. It is useful in case of making simulation analysis for specific regions.
  • T-fasta: The normal pipeline will use transposed fasta format (.tfa). This format contains the sequences for each individual in vertical instead of horizontal. It also includes a header with the names of each sequence and the number position of each nucleotide at the beginning of each row.
