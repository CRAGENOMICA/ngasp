About

Welcome to ngasp, the free, open source computational solution for performing next generation analysis of sequence polymorphisms using NGS data.

ngasp can be used under the LGPL version 2.1 license. Find out the entire source code on GitLab.

For more information, visit bioinformatics.cragenomica.es/projects/ngaSP.

Installation Requirements

Docker:

Install Docker from https://docs.docker.com/engine/installation

Example of Installation on Centos 7:

$ sudo yum update
$ tee /etc/yum.repos.d/docker.repo <<-'EOF'
[dockerrepo]
name=Docker Repository
baseurl=https://yum.dockerproject.org/repo/main/centos/7/
enabled=1
gpgcheck=1
gpgkey=https://yum.dockerproject.org/gpg
EOF
$ sudo yum install docker-engine   or    sudo yum install docker
$ sudo systemctl enable docker.service
$ sudo systemctl start docker
$ sudo groupadd docker
$ sudo usermod -aG docker $USER
$ sudo newgrp docker

Docker Compose:

Install Docker Compose from https://docs.docker.com/compose/install

Example of Installation on Centos 7:

$ curl -L https://github.com/docker/compose/releases/download/1.9.0-rc1/docker-compose-`uname -s`-`uname -m` > /usr/bin/docker-compose
$ chmod +x /usr/bin/docker-compose


Installation

Installation steps:

• Uncompress the downloaded ngasp's package.

• Open a console

$ cd <ngasp_folder>
$ docker-compose -f compose.yml build
$ docker-compose -f compose.yml up


Getting Started

Open your web navigator and access the ngasp's front-end at http://localhost:3000. You will see see a web page with a wellcome message in the center and some links on the left.

Navigate through the links on the left and you'll access the user manuals, the Experiment Editor and the Pipeline Editor.

Select the Experiment Editor and start creating your experiments just doing right click at the empty experiment document and adding calculation nodes.

For more information on how to use ngasp, check out the support page.
