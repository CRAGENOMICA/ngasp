Calculation List


CALCULATION NAME DESCRIPTION
BAM to m/pileup Previous conversion necessary to call SNPs, calculate frequencies and other posterior analysis.
GH Caller Call SNPs and estimate frequencies using Roesti (2012) algorithm.
GH caller is used instead other more popular SNP callers because it calculates unbiasedly the frequencies of the variants as well as the number of samples truly sequenced. The aim of this algorithm is to have accurate estimates of the frequency of the variants and not the highest sensibility to find SNPs. This algorithm allows to estimate accuaretly the levels and patterns of variability (Nevado et al 2014).
Fasta Converter Conversion from fasta to tfasta files. Filter positions to be studied.
fastaconvtr is a tool for converting fasta files to a more useful format similar to gVCF. This format contains haploid, diploid, polyploid or pooled data in a easy and simply format. fastaconvtr also filter functional regions from GTF files and allow masking regions of the genome.
Mstatspop Calculation of a large number of population genetics statistics related to the level and frequency of the variants.
mstatspop is designed to analyze genome sequence data containing missing information. It estimates the level and patterns of variability from several populations and containing outgroup species. It performs sliding window analysis for physical or for functional positions.
Collect Data Columns Script that select a number of statistics for posterior analysis.
